TimeSeriesCollection Class¶

class TimeSeriesCollection(iterable=None)[source]¶

Bases: collections.abc.MutableSet

Custom OrderedSet for TimeSeries objects.

This container remembers the order in which its members were added, so that every entry has a numeric index that can be looked up.

Add ts and return its index.

Parameters:ts (TimeSeries) – A TimeSeries to add.
Returns:Index of the added TimeSeries.
Return type:int, None

Note

If a TimeSeries with the same ts_name is already in the container, the method does not modify the TimeSeriesCollection and returns the index associated with the previously existing TimeSeries.
